# End My Day Output Formats

## Default: Full End-of-Day Review

```
End of Day — [Day, Date]

FOCUS THIS WEEK
| Day | Focus hrs | Status |
|-----|-----------|--------|
| Mon | [X]h      | ✓      |
| Tue | [X]h      | ✓      |
| Wed | [X]h      | ⚠ at risk |
| Thu | [X]h      | 🔴 <2h |
| Fri | [X]h      | ✓      |
Week total: [X]h — [✓ on track (15–20h target) / ⚠ below 15h / ✓ week complete: Xh]

-- if fixes needed: --
Proposed fixes:
1. Move "DEEP WORK TIME" Thu 2–3pm → Thu 4–5pm (conflicts with Design Review)
2. Add focus block Fri 9–11am (open slot — 2h toward weekly target)
3. Decline "Quick sync" Wed 3–3:30pm *(you're optional)*
Approve all, pick numbers, or skip?
-- if healthy: --
All days protected. Week on track.

TOMORROW — [Day, Date]
| Time | Event |
|------|-------|
| [Start]–[End] | [Event title] |
| [Start]–[End] | [Event title] *(you're optional)* |

A few things to note:
- You have a free block from [X]–[Y] ([duration])
- Your [morning/afternoon] is back-to-back from [X] to [Y]
- [Meeting] looks under-prepped — consider [prep suggestion] before tomorrow

ACTION ITEMS  (from today's meetings)
1. [Action item] — [Meeting name]
2. [Action item] — [Meeting name]
3. ...
Add all to TASKS.md, pick numbers, or skip?

INSIGHTS
- [Observation worth keeping] — [Meeting name]  → suggest: knowledge/domains/[domain]
- [Pattern noticed across meetings]
- ...
File any to knowledge/domains? (say which, or skip)

Ready to wrap up?
- Say "approve [n,n,n]" to accept specific proposals above
- Say "approve all" to accept everything at once
- Say "/sync-meetings" if today's notes look incomplete
- Say "/start-my-day" tomorrow morning
```

## Friday Variant (week complete)

Same structure but FOCUS THIS WEEK shows final actuals, not projections:

```
FOCUS THIS WEEK  (final)
| Day | Focus hrs | Status |
...
Week total: [X]h  [✓ hit target / ⚠ came up short — Xh]
```
